# Sistema de Gestión Parlamentaria


Sistema web para digitalizar procesos administrativos en un entorno municipal.

## 🚀 Tecnologías
- Java (Spring Boot)
- MySQL
- Docker / Docker Compose

## ⚙️ Cómo ejecutar

docker-compose up --build

## 🧠 Funcionalidades
- Gestión de expedientes
- Control de roles
- Seguimiento de estados
